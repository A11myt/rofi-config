# Rofi Konfiguration

Dieses Verzeichnis enthält meine vollständige Rofi-Konfiguration für Linux. Alle Einstellungen, Farbschemata und Layouts sind in der Datei `config.rasi` zusammengefasst.

## Datei

- `config.rasi`: Zentrale Konfigurationsdatei für Aussehen, Farben und Verhalten von Rofi.

## Installation

1. Kopiere die Datei `config.rasi` nach `~/.config/rofi`.
2. Stelle sicher, dass Rofi installiert ist (`sudo apt install rofi`).
3. Starte Rofi mit `rofi -show drun`.

## Anpassung

Bearbeite `config.rasi`, um Farben, Schriftarten und Layout nach deinen Wünschen zu ändern.
